LEMA: LMT-oriented Entity Mapping Network for Long-text Composed Image Retrieval

Zixu Li1, Zhiheng Fu1, Yupeng Hu1, Zhiwei Chen1, Xuemeng Song1, Weili Guan2,
1Shandong University,2Harbin Institute of Technology (Shenzhen)
Date: 2024-11
Code L-CIR Dataset

Abstract

Composed Image Retrieval (CIR) is a significant paradigm in image retrieval that enables users to locate target images through a multimodal query, comprising a reference image and modification text. In this framework, the reference image conveys the user's fundamental retrieval requirements, while the modification text articulates desired alterations to the reference image. However, the modification texts in existing CIR datasets are typically brief and insufficient to accommodate users' sophisticated modification intentions. To address this limitation, we introduce Long-text Composed Image Retrieval (L-CIR), which facilitates detailed modification-based retrieval intentions through long modification text (LMT). Meanwhile, we extend the classical CIR datasets (FashionIQ and CIRR) to provide sufficient data support for improving the model's understanding of LMT. Furthermore, we present the first L-CIR model, the LMT-oriented Entity Mapping Network (LEMA), designed to address two critical challenges in L-CIR: 1) entity ignoring, and 2) one-to-many correspondences. This model establishes a robust baseline for the L-CIR task. Our evaluation encompasses multiple representative CIR baselines trained on the newly constructed L-CIR dataset. Extensive experimental results demonstrate that our proposed LEMA model achieves superior retrieval performance in the L-CIR task while maintaining an optimal balance between retrieval accuracy and computational efficiency.
